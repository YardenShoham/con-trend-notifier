setPreference

  • setPreference(preferenceRequest: SetPreferenceDto, req: AuthorizedRequest, res: Response): Promise<Response>
    • Defined in src/controllers/preferenceController.ts:111

    Sets a user's wanted probability threshold to be notified about a rise/fall of a symbol.

    Parameters

    • preferenceRequest: SetPreferenceDto

      The object containing the symbol information and wanted probability.

    • req: AuthorizedRequest

      The Express request + jwt payload.

    • res: Response

      The Express response.

    Returns Promise<Response>

    One of the following:

    • Nothing if everything went well. Status: NO_CONTENT.
    • A { error: string } object if the user does not exist or the probability is not between -1 and 1. Status: UNPROCESSABLE_ENTITY.
    • A ValidationError[] if one or more assets were not valid (must be uppercase). Status: UNPROCESSABLE_ENTITY.
